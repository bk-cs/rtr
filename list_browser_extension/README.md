## DESCRIPTION
List installed extensions for Chromium-based (Chrome, Edge) browsers

## EXAMPLES

### REAL-TIME RESPONSE
```
runscript -CloudFile="list_browser_extension"
```
### PSFALCON
```powershell
PS>Invoke-FalconRtr runscript "-CloudFile='list_browser_extension'" -HostId <id>, <id>
```
