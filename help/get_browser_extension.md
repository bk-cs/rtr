## DESCRIPTION
List installed extensions for Chromium-based (Chrome, Edge) browsers

## PLATFORMS
Windows

## PARAMETER Filter
Restrict list using a RegEx pattern

## PARAMETER Log
Save results within a Json file in the Rtr directory
