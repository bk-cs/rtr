## DESCRIPTION
List FileVersionInfo for Portable Executable (PE) file

## PLATFORMS
Windows

## PARAMETER Path
Path of the PE file

## PARAMETER Log
Save results within a Json file in the Rtr directory

### REAL-TIME RESPONSE
