## DESCRIPTION

## PLATFORMS
Mac, Linux, Windows

## PARAMETER SensorTag
One or more SensorTag values (Required)

## PARAMETER Log
Save results within a Json file in the Rtr directory

## EXAMPLES

### REAL-TIME RESPONSE
```
runscript -CloudFile="remove_sensortag" -CommandLine=```'{"SensorTag":["my","example","tags"]}'```
```
