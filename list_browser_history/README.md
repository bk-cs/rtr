## DESCRIPTION
List history of visited domains for Chromium-based (Chrome, Edge) browsers

## EXAMPLES

### REAL-TIME RESPONSE
```
runscript -CloudFile="list_browser_history"
```
### PSFALCON
```powershell
PS>Invoke-FalconRtr runscript "-CloudFile='list_browser_history'" -HostId <id>, <id>
```
